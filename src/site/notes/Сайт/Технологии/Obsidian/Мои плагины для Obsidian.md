Список плагинов, которые я написала для приложения [[Сайт/Технологии/Obsidian/Obsidian\|Obsidian]].

## Официальные

Эти плагины официально добавлены в Обсидиан и могут быть установлены через браузер плагинов.

- <span class="custom-list-icon" data-symbol="^" style="color: rgb(var(--color-yellow-rgb));"><svg   xmlns="http://www.w3.org/2000/svg" width="24" height="24" viewBox="0 0 24 24" fill="none"   stroke="currentColor" stroke-width="2" stroke-linecap="round" stroke-linejoin="round" class="svg-icon   lucide-star"><polygon points="12 2 15.09 8.26 22 9.27 17 14.14 18.18 21.02 12 17.77 5.82 21.02 7 14.14 2   9.27 8.91 8.26 12 2"></polygon></svg></span> [Pretty Properties](https://github.com/anareaty/pretty-properties)
    - Позволяет добавлять в заметку баннеры, обложки и боковые изображения рядом с блоком свойств
    - Добавляет меню для выбора цвета свойств и тэгов
    - Позволяет скрывать ненужные или пустые свойства
    - Позволяет установить свой формат даты
    - Множество других настроек для отображения и форматирования свойств
- [HTML checkboxes](https://github.com/anareaty/html-checkboxes)
    - Упрощает добавление в заметку чекбоксов в формате HTML
    - Делает HTML-чекбоксы кликабельными
    - <span class="custom-list-icon" data-symbol="!!" style="color: rgb(var(--color-red-rgb));"><svg   xmlns="http://www.w3.org/2000/svg" width="24" height="24" viewBox="0 0 24 24" fill="none"   stroke="currentColor" stroke-width="2" stroke-linecap="round" stroke-linejoin="round" class="svg-icon   lucide-alert-triangle"><path d="m21.73 18-8-14a2 2 0 0 0-3.48 0l-8 14A2 2 0 0 0 4 21h16a2 2 0 0 0 1.73-3Z"></path><path d="M12 9v4"></path><path d="M12 17h.01"></path></svg></span> У плагина есть проблемы с производительностью и он больше не обновляется
- [Callout menu](https://github.com/anareaty/callout-menu)
    - Добавляет дополнительные функции в меню, которое открывается при правом клике на выносной блок
    - Позволяет добавить в меню свои типы выносных блоков для быстрого переключения

## На проверке

Эти плагины пока проверяются командой Obsidian и будут добавлены в браузер плагинов после утверждения. До этого момента могут быть установлены вручную или с помощью плагина BRAT.

- [Task Count](https://github.com/anareaty/task-count)
    - Автоматически подсчитывает количество выполненных или невыполненных задач в заметке и сохраняет это число в свойствах
    - В том числе поддерживает задачи из плагина TaskNotes

## Неофициальные

Эти плагины написаны для личного пользования, и я пока не планирую официально добавлять их в Обсидиан. Устанавливаются вручную или с помощью плагина BRAT.

> [!warning] Предупреждение
> Неофициальные плагины могут содержать баги, жёстко прописанные настройки или просто быть недописаны или заброшены. Я не обеспечиваю поддержку этих плагинов и не принимаю запросов на их улучшение. Устанавливайте на свой страх и риск!

- [Custom Lists](https://github.com/anareaty/custom-lists)
    - Клон плагина List Callouts (но со своими особенностями)
    - Добавляет иконки и цвета к элементам списка
    - Поддерживаются только иконки Lucide
    - Цвет иконки, фона и текста можно настроить отдельно
- [Checkboxes Plus](https://github.com/anareaty/checkboxes-plus)
    - Позволяет добавлять чекбоксы в таблицы Markdown
    - Добавляет меню для выбора статуса при правом клике на чекбокс
    - Добавляет значки для альтернативных чекбоксов
    - Поддерживается только четыре альтернативных статуса чекбоксов без возможности редактирования
- [Clean Syncthing Conflicts](https://github.com/anareaty/clean-syncthing-conflicts)
    - Находит все конфликтные файлы Syncthing и переносит их в папку `.sync-conflicts`
    - Папка периодически очищается
- [Note Classes](https://github.com/anareaty/note-classes)
    - Вводит специальные шаблоны классов, позволяющие изменить свойства файла, не затрагивая содержимое
    - Изменение класса файла удаляет из него все ненужные свойства и добавляет недостающие
    - При изменении шаблона можно автоматически обновить свойства во всех файлах с этим классом
    - Есть опция для автоматического перемещения всех файлов с определённым классом в определённую папку
- [Useful commands](https://github.com/anareaty/useful-commands)
    - Добавляет множество небольших, но полезных команд для Обсидиана
- [Views](https://github.com/anareaty/views)
    - Плагин для создания визуально редактируемых таблиц (написан до появления баз данных в Обсидиане)
    - Для работы требуется Dataview
    - Может работать на старых версиях Обсидиана, не поддерживающих базы данных
- [Writing System](https://github.com/anareaty/writing-system)
    - Плагин для организации писательских проектов и отслеживания писательского прогресса
    - В разработке (значительная часть функционала пока отсутствует)
    - Нестандартная организация проектов (с помощью ссылок, а не папок)

## Модификации чужих плагинов

Это форки плагинов, написанных другими людьми, с небольшими персональными изменениями. В них могут быть удалены или добавлены какие-то функции, изменены стили и т. п. Эти форки не будут официально публиковаться или поддерживаться, и в целом я рекомендую пользоваться оригинальными плагинами, если только вы не склонны к экспериментам. 

- [Форк Horizontal Blocks](https://github.com/anareaty/horizontal-blocks-fork)
    - Упрощены стили и удалены настройки
    - Добавлена поддержка кликабельных чекбоксов
    - Изменён способ сохранения ширины колонок (ширина сохраняеся в самом блоке кода вместо файла настроек)
- [Форк Journal](https://github.com/anareaty/obsidian-journal-fork)
    - Изменены стили и упрощён внешний вид календаря

> [!info] Все мои плагины написаны без использования ИИ.
